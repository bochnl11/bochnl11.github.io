Nashville Housing Data

This SQL project focuses on cleaning and standardizing Nashville housing market data to prepare it for analysis. The script performs several key data cleaning operations including standardizing sale dates, filling in missing property addresses through parcel ID matching, breaking down both property and owner addresses into distinct components (street, city, state), standardizing categorical values in the "Sold as Vacant" field, and identifying and managing duplicate records. The project demonstrates advanced SQL techniques including Common Table Expressions (CTEs), window functions, and various string manipulation methods to transform raw housing data into a clean, structured format suitable for analysis. The final result is a refined dataset with standardized formats, no missing values, and no duplicates.

Global COVID Infection Data

This SQL project analyzes global COVID-19 data to track infection rates, death rates, and vaccination progress across different countries and continents. The analysis includes creating specialized tables for death statistics and vaccination data, calculating key metrics such as infection rates relative to population, tracking vaccination progress over time, and examining correlations with demographic and socioeconomic factors. The analysis spans multiple aspects of the pandemic, from basic case tracking to sophisticated population-adjusted metrics, while also incorporating factors like GDP per capita, population density, and healthcare infrastructure to provide deeper insights into the pandemic's global impact.
