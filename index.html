Predictive Modeling

This Python project analyzes Bundesliga soccer data from StatsBomb to generate comprehensive player ratings and performance predictions. The analysis processes match events, player statistics, and team data to create a sophisticated rating system that weights different actions (passes, shots, interceptions, etc.) based on player positions. Using a Random Forest model, the project generates predictive ratings for players while accounting for their roles and performance metrics. The script includes data processing from JSON files, calculation of advanced statistics, and machine learning implementation to evaluate and predict player performance. The final output includes both actual performance ratings and predicted ratings, allowing for comparison and identification of over/underperforming players in the 2023/2024 season.

Nashville Housing Data

This SQL project cleans and standardizes Nashville housing market data for analysis. It includes operations like standardizing sale dates, filling missing property addresses using parcel IDs, breaking down property and owner addresses, standardizing “Sold as Vacant” values, and handling duplicates. The project employs advanced SQL techniques like CTEs, window functions, and string manipulation to transform raw data into a clean, structured format. The final result is a refined dataset free of missing values and duplicates, ready for analysis.

Movie Data in Jupyter

This Python project written and executed in a Jupyter Notebook examines a historical movie dataset to uncover insights about the film industry. The analysis involves data cleaning, including handling missing values and converting data types, as well as exploring relationships between variables like budget, gross earnings, and ratings. World Happiness Data in R

This R script analyzes World Happiness Report 2024 data to investigate factors influencing global happiness scores through statistical modeling and visualization. The analysis combines correlation studies, linear regression modeling, and temporal trend analysis to understand what drives happiness across different countries and regions. The project examines patterns across selected nations, including the United States, Canada, Germany, Brazil, and Australia, while also providing continental comparisons and exploring relationships between happiness and economic indicators like GDP. Using a variety of statistical techniques including correlation heatmaps, residual analysis, and time series visualization, the code delivers comprehensive insights into global happiness patterns while ensuring data quality through thorough cleaning and validation processes.
